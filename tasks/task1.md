## Технічне Завдання: Сайт для керування списком товарів

### Загальна суть задачі

Створи простий сайт, який дозволяє користувачу керувати списком товарів з можливістю фільтрації за назвою. Використовуй `useMemo` для оптимізації фільтрації, та власний хук для відкриття і закриття модального вікна. Користувач повинен мати можливість додавати, видаляти та фільтрувати товари.

### Структура компонентів

#### 1. `App`

**Опис:** Головний компонент додатку.
**Призначення:** Містить логіку керування списком товарів та фільтрації.

#### 2. `ProductList`

**Опис:** Компонент для відображення списку товарів.
**Призначення:** Відображає список товарів на основі переданих пропсів.
**Пропси:**

- `products` (масив) – масив об'єктів товарів для відображення.
- `onDelete` (функція) – функція для видалення товару.

#### 3. `ProductItem`

**Опис:** Компонент для відображення одного товару.
**Призначення:** Відображає інформацію про товар та кнопку для видалення.
**Пропси:**

- `product` (об'єкт) – об'єкт товару.
- `onDelete` (функція) – функція для видалення товару.

#### 4. `SearchBar`

**Опис:** Компонент форми для введення ключових слів пошуку.
**Призначення:** Містить поле введення для фільтрації товарів.
**Пропси:**

- `onSearch` (функція) – функція, яка буде викликана при зміні значення в полі введення.

#### 5. `AddProductForm`

**Опис:** Компонент форми для додавання нового товару.
**Призначення:** Містить поле введення для назви товару та кнопку для додавання товару.
**Пропси:**

- `onAdd` (функція) – функція, яка буде викликана при додаванні нового товару.

#### 6. `Modal`

**Опис:** Компонент модального вікна.
**Призначення:** Відображає модальне вікно з вмістом.
**Пропси:**

- `isOpen` (булевий) – визначає, чи відкрите модальне вікно.
- `onClose` (функція) – функція для закриття модального вікна.
- `children` (jsx) – вміст модального вікна.

#### 7. `useModal`

**Опис:** Власний хук для керування станом модального вікна.
**Призначення:** Забезпечує логіку відкриття та закриття модального вікна.

### Компонент `App`

#### Стан:

- `products` (масив) – зберігає список товарів.
- `searchQuery` (рядок) – зберігає значення пошукового запиту.
- `isModalOpen` (булевий) – зберігає стан модального вікна.

#### Призначення:

1. Відображає компонент `SearchBar` для введення ключових слів пошуку.
2. Використовує `useMemo` для фільтрації списку товарів на основі `searchQuery`.
3. Відображає компонент `ProductList` для відображення списку товарів.
4. Використовує компонент `Modal` для додавання нового товару.
5. Відображає компонент `AddProductForm` всередині `Modal`.

#### Функції:

- `handleSearch` (функція) – оновлює стан `searchQuery`.
- `handleAddProduct` (функція) – додає новий товар до списку.
- `handleDeleteProduct` (функція) – видаляє товар зі списку.

### Компонент `SearchBar`

#### Стан:

- `searchTerm` (рядок) – зберігає значення поля введення.

#### Призначення:

1. Відображає поле введення для пошуку товарів.

#### Пропси:

- `onSearch` (функція) – функція для передачі значення пошуку до компонента `App`.

#### Функції:

- `handleInputChange` (функція) – обробляє зміну значення в полі введення.

### Компонент `AddProductForm`

#### Стан:

- `productName` (рядок) – зберігає значення поля введення назви товару.

#### Призначення:

1. Відображає поле введення для назви товару та кнопку для додавання.

#### Пропси:

- `onAdd` (функція) – функція для передачі нового товару до компонента `App`.

#### Функції:

- `handleInputChange` (функція) – обробляє зміну значення в полі введення.
- `handleSubmit` (функція) – обробляє натискання кнопки "Додати", викликає функцію `onAdd`.

### Компонент `ProductList`

#### Призначення:

1. Відображає список товарів.

#### Пропси:

- `products` (масив) – масив об'єктів товарів.
- `onDelete` (функція) – функція для видалення товару.

### Компонент `ProductItem`

#### Призначення:

1. Відображає інформацію про товар та кнопку для видалення товару.

#### Пропси:

- `product` (об'єкт) – об'єкт товару.
- `onDelete` (функція) – функція для видалення товару.

### Компонент `Modal`

#### Призначення:

1. Відображає модальне вікно з вмістом.

#### Пропси:

- `isOpen` (булевий) – визначає, чи відкрите модальне вікно.
- `onClose` (функція) – функція для закриття модального вікна.
- `children` (jsx) – вміст модального вікна.

### Хук `useModal`

#### Призначення:

1. Забезпечує логіку відкриття та закриття модального вікна.

#### Повертає:

- `isOpen` (булевий) – стан модального вікна.
- `openModal` (функція) – функція для відкриття модального вікна.
- `closeModal` (функція) – функція для закриття модального вікна.

### Функціональні вимоги

1. **Додавання товарів:** Користувач повинен мати можливість додавати нові товари через модальне вікно.
2. **Видалення товарів:** Користувач повинен мати можливість видаляти товари зі списку.
3. **Фільтрація товарів:** Користувач повинен мати можливість фільтрувати товари за назвою через поле пошуку.

### Підказки

1. Використовуй `useState` для зберігання стану списку товарів, пошукового запиту та стану модального вікна.
2. Використовуй `useMemo` для оптимізації фільтрації списку товарів на основі `searchQuery`.
3. Використовуй власний хук `useModal` для керування модальним вікном.
